## sum()

#### Explanation

If you want to get sum value of number array, use sum function

#### Parameter

1. paramName: arr, type: Number Array, Optional: false

#### Return

return Number

#### usage

```js
import DataDancer from 'dataDancer';

const d = new DataDancer.NumberMethods();

const numArr = [1,3,5];
const rs = d.sum(numArr);

console.log(rs); // 9
```
